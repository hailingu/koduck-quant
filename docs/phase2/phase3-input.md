## 关键决策点

### Phase 3 需要决策的问题

1. **测试数据库选择**
   - 选项 A: 继续使用 H2 (快速但不够真实)
   - 选项 B: Testcontainers PostgreSQL (慢但更真实)
   - 选项 C: 混合策略 (单元用 H2，集成用 PostgreSQL)

2. **PMD 治理策略**
   - 选项 A: 集中 2 周完成 (风险高，回归风险)
   - 选项 B: 分批 6 周完成 (风险低，持续改进)

3. **性能测试频率**
   - 选项 A: 每次 PR 都跑 (慢但及时)
   - 选项 B:  nightly 运行 (快但延迟)
   - 选项 C: 手动触发 (灵活但可能遗漏)
