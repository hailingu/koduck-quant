## 4. 优化建议

### 4.1 工程实践改进

| 建议 | 现状 | 目标 | 实施方式 |
|------|------|------|----------|
| **pre-commit hook** | 无 | 强制质量检查 | 配置 `make quality` 自动运行 |
| **测试数据工厂扩展** | 基础 | 完善 | 补充更多 Entity Builder |
| **CI 并行度提升** | 串行 | 并行 | 单元/集成测试并行执行 |
| **缓存监控指标** | 无 | 有 | Micrometer + Prometheus |
| **慢查询告警** | 无 | 有 | PostgreSQL 慢查询日志 + 告警 |

| PR 覆盖率增量 | 无 | >= 60% | 立即实施 |
| PMD 全量 | 计划 | 执行 | P4-04 完成后 |
| SonarQube 质量阈 | 基础 | 严格 | Phase 4 中期 |

### 4.3 开发体验改进

| 建议 | 说明 |
|------|------|
| IDE 统一配置 | 提供 VSCode + IntelliJ 代码格式化配置 |
