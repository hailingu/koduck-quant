# Architecture Decision Records (ADR) - Frontend

本文档目录存放 `koduck-frontend` 的架构决策记录（ADR）。

## 编号建议

文件命名格式：`NNNN-title.md`

- `0000`: 模板
- `2000-2999`: 前端架构决策（推荐）

示例：`2001-state-management.md`

## 状态建议

- `提议 (Proposed)`
- `已接受 (Accepted)`
- `已拒绝 (Rejected)`
- `已弃用 (Deprecated)`
- `已被替代 (Superseded)`

## 使用方式

1. 复制模板：
   - `cp docs/adr/0000-template.md docs/adr/NNNN-title.md`
2. 填写背景、备选方案、决策结果与影响
3. 在 PR 中与代码一起提交
